# Playwright E-Commerce Automation Framework

## Tech Stack

- Playwright
- JavaScript
- POM Framework
- API Testing
- Jenkins
- GitHub Actions
- HTML Reporting

---

## Features

- Registration Automation
- Login Automation
- Product Search
- Add To Cart
- Update Quantity
- Checkout
- Order Confirmation
- API Validation
- Parallel Execution
- Retry Logic
- Screenshot Capture
- Video Capture
- Trace Capture

---

## Installation

```bash
npm install
```

---

## Install Browsers

```bash
npx playwright install
```

---

## Run All Tests

```bash
npx playwright test
```

---

## Run Specific Test

```bash
npx playwright test tests/login.spec.js
```

---

## Run On Chrome

```bash
npx playwright test --project=chromium
```

---

## Run On Firefox

```bash
npx playwright test --project=firefox
```

---

## Open Report

```bash
npx playwright show-report
```

---

## Framework Design Pattern

- Page Object Model
- Data Driven Testing
- Reusable Utilities
- API + UI Validation
- Cross Browser Execution
